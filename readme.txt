== Description ==

Enefti NFT Marketplace Core lite is the most compact and easy to use NFT Marketplace plugin, developed by one of bestselling premium WordPress theme/plugin authors, <a href="https://modeltheme.com/" target="_blank">ModelTheme</a>. The free version of the plugin comes prepackaged with features to start an NFT Marketplace in WordPress, in just a few minutes. The plugin supports multiple blockchains at the same time, multiple blockchain standards and it is fast as lightning.

== Frequently Asked Questions ==

= How do I install Enefti NFT Marketplace Core lite? =

There are two ways you can install Enefti NFT Marketplace Core lite plugin.

You can install the plugin from your WordPress dashboard:
1. Navigate to your WordPress Dashboard -> Plugins -> Add New
2. Search for Enefti NFT Marketplace Core lite plugin
3. Install and Activate Enefti NFT Marketplace Core lite from your Plugins page.

Or you can install it directly from WordPress.org
1. Locate and download Enefti NFT Marketplace Core lite on wordpress.org
2. Upload the nft-marketplace-core-lite directory to your ‘/wp-content/plugins/’ directory using your preferred method (ftp, sftp, scp, etc…)
3. Install and Activate Enefti NFT Marketplace Core lite plugin from your Plugins page.

= How do I get support? =

If you require assistance while using the Enefti NFT Marketplace Core lite plugin, you can visit the <a href="https://wordpress.org/support/plugin/nft-marketplace-core-lite/" target="_blank">support forum</a> on WordPress.org, or you can <a href="https://modeltheme.ticksy.com" target="_blank">submit a ticket</a> and our support agents will respond to you as soon as possible.

= Do I need to know how to code? =

Absolutely not. Enefti NFT Marketplace Core lite was designed to allow you to create NFTs directly from the visual dashboard of your WordPress, without the need to understand or use any coding.

= What if I update to the Premium version? =

In the event that you decide to upgrade to the <a href="https://modeltheme.com/portfolio/enefti-nft-marketplace-core/" target="_blank">Premium version</a> of Enefti NFT Marketplace Core lite you’ll get more than 40+ premium features to your NFT Marketplace, including: buy, sell, resell NFTs on multiple blockchains.

= Does Enefti NFT Marketplace Core lite plugin work with any WordPress theme? =

Yes. Enefti NFT Marketplace Core lite is designed to work with all WordPress themes, but our recommendation is the dedicated <a href="https://modeltheme.com/portfolio/enefti-nft-marketplace-theme/" target="_blank">Enefti Theme</a>.

= What is the <a href="https://modeltheme.com/portfolio/enefti-nft-marketplace-theme/" target="_blank">Enefti Theme</a> and what is its connection to Enefti NFT Marketplace Core lite plugin? =

The <a href="https://modeltheme.com/portfolio/enefti-nft-marketplace-theme/" target="_blank">Enefti Theme</a> was developed by the same team that built Enefti NFT Marketplace Core lite and it comes with 10+ NFT Related website demos. While Enefti NFT Marketplace Core lite work with any theme, you will certainly get the most out of the plugin when using it together with the Enefti Theme, as the plugin was primarily developed to work best with it.

== Installation ==

= Minimum Requirements =

* WordPress 5.0 or greater
* PHP version 5.6 or greater
* MySQL version 5.0 or greater

= We recommend your host supports: =

* PHP version 7.0 or greater
* MySQL version 5.6 or greater
* WordPress Memory limit of 128 MB or greater

= Installation =

From your WordPress dashboard

1. Go to your WordPress Dashboard -> Plugins -> Add New
2. Search for Enefti NFT Marketplace Core lite
3. Activate Enefti NFT Marketplace Core lite from your Plugins page.
4. Enjoy :)

From wordpress.org

1. Download Enefti NFT Marketplace Core lite.
2. Upload the nft-marketplace-core-lite directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc…)
3. Activate Enefti NFT Marketplace Core lite from your Plugins page.
4. Enjoy :)
